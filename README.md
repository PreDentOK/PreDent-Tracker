# PreDent Connect - Hours Tracker

A responsive web application for Pre-Dental students to track their shadowing and volunteering hours.

## Features
- **Hours Tracking:** Log Shadowing and Volunteer hours with dates, locations, and doctor names.
- **Visual Stats:** Persistent progress rings and visual breakdowns of dental vs. non-dental experience.
- **Local Storage:** Data is saved to the browser automatically.
- **Leaderboard:** Optional community ranking system powered by Firebase.
- **Export:** Download all logs as a CSV file.

## Setup
1. Clone this repository or download the files.
2. Open `index.html` in your browser to run the app locally.

## Deployment
This app is ready for GitHub Pages.
1. Push the files to your repository main branch.
2. Go to Settings > Pages > Select Main Branch > Save.
